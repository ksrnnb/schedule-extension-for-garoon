A Chrome extension that enhances Garoon's event notifications.
Get desktop reminders so you never miss a meeting.

[Features]

- Pre-meeting desktop notifications
  Shows a desktop notification a configurable number of minutes before each event.
  An optional chime sound is supported.

- Next event in the toolbar badge
  The toolbar icon's badge shows the start time of the next event today.

- Today's schedule popup
  Clicking the toolbar icon opens a popup listing today's events,
  marked as finished, in progress, or upcoming.

[How to use]

1. Install the extension.
2. Open the options page and enter your Garoon URL (e.g. https://example.cybozu.com/g/).
3. Stay logged in to Garoon in your browser; events sync automatically.

[Supported environment]

Only Garoon Cloud (*.cybozu.com) is supported.
On-premises Garoon is not supported.

[Privacy]

- Fetched event data is stored only in your browser's local storage.
- No data is ever sent to any external server.
- Only Garoon's official REST API is used.
