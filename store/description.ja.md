Garoon の予定通知を強化する Chrome 拡張機能です。
デスクトップ通知で予定を見逃しません。

【主な機能】

- 予定の事前通知
  指定した分数前にデスクトップ通知を表示します。任意で通知音（ベル音）も鳴らせます。

- 次の予定をツールバーに表示
  ツールバーアイコンのバッジに、今日の次の予定の開始時刻が表示されます。

- 今日の予定一覧
  ツールバーアイコンをクリックすると、今日の予定が一覧表示されます。
  終日予定や、終了済み / 進行中 / 未開始の状態を区別して表示します。

【使い方】

1. 拡張機能をインストールします。
2. オプションページを開き、ご利用の Garoon の URL（例: https://example.cybozu.com/g/）を入力して保存します。
3. Garoon にブラウザでログインしておくと、自動的に予定が同期されます。

【対応環境】

サイボウズのクラウド版 Garoon (*.cybozu.com) のみ対応しています。
オンプレミス版 Garoon には対応していません。

【プライバシーについて】

- 取得した予定データはブラウザのローカルストレージにのみ保存されます。
- 外部サーバーへのデータ送信は一切行いません。
- 通信は Garoon 公式の REST API のみを使用します。
